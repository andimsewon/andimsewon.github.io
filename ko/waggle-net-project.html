WaggleNet – LoRa 및 MQTT 기반 내부·외부 벌통 환경 모니터링 시스템

클릭하여 논문 보기

이 프로젝트는 양봉 환경에서 내부와 외부 조건을 동시에 모니터링하는 저비용·확장형 IoT 시스템을 구축하기 위해 진행되었다. Purdue University에서 2025년 여름, 국제 연구 장학생으로 참여하여 스마트 양봉(Smart Beekeeping) 분야의 IoT 기반 환경 감시 시스템을 설계하고 구현하였다.

기존의 스마트 벌통 시스템은 내부 온도와 습도만 감지하거나, 고가의 GSM/LTE 통신 장비에 의존해 확장성과 실시간성이 떨어지는 한계가 있었다. 이를 개선하기 위해, Wi-Fi LoRa 32 (V3) 모듈을 사용하여 내부·외부 양방향 환경 데이터를 수집하는 워커 노드(Worker Node)와 MQTT를 통해 클라우드 서버로 데이터를 전송하는 마스터 노드(Master Node)를 설계하였다. 각 노드는 온도·습도(DHT22), 조도(LDR), GPS 모듈을 통합하여 3분 간격으로 데이터를 수집하고, JSON 포맷으로 변환 후 LoRa 통신을 통해 송신한다.

수집된 데이터는 MQTT 서버와 SQLite 기반 클라우드 서버에 저장되어, Flutter로 개발된 모바일 앱을 통해 시각화된다. 사용자는 앱에서 실시간으로 벌통의 온도, 습도, 조도, GPS 위치를 확인할 수 있으며, 이상치 발생 시 Firebase Cloud Messaging 기반 긴급 알림 시스템이 자동으로 경고를 보낸다.

본 시스템은 Purdue University의 실험 농장과 건물 외부에서 실험되었으며, LoRa 통신은 개방 환경에서 최대 110m, 장애물 환경에서 95m까지 안정적으로 작동하였다. 내부 벌통에 설치된 센서 또한 정확히 데이터를 전송해, 목재 구조 내부에서도 LoRa의 신뢰성을 검증하였다.

결과적으로 WaggleNet은 저비용(모듈당 $10~15), 소형 설계, 모듈형 확장성, 모바일 접근성을 모두 만족하는 시스템으로, 스마트 양봉 및 스마트 농업 환경에서의 실시간 환경 감시에 활용 가능함을 입증하였다. 향후 계획으로는 AI 기반 이상 감지(Anomaly Detection), 하드웨어 소형화, 자율적 데이터 라우팅 기능을 통합하여 완전한 지능형 양봉 관리 플랫폼으로 발전시킬 예정이다.

기술 스택: 하드웨어: ESP32-S3 (Heltec Wi-Fi LoRa 32 V3), DHT22, LDR, GPS (NEO-6M) | 소프트웨어: C++, Python, MQTT, SQLite, Flutter, Firebase | 통신 프로토콜: LoRa (433/868/915MHz), MQTT Publish/Subscribe
