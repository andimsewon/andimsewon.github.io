
# 두목 Personal CV Website (GitHub Pages)

이 저장소는 **정적 HTML/CSS**로 만든 원페이지 포트폴리오입니다. 빌드 과정 없이 바로 GitHub Pages에서 호스팅됩니다.

## 배포
1. 저장소 이름을 `<your-username>.github.io` 로 생성합니다.
2. 이 레포의 파일들을 그대로 업로드(또는 커밋/푸시)합니다.
3. 브라우저에서 `https://<your-username>.github.io` 접속.

## 커스터마이징
- `assets/profile.png`: 프로필 사진을 이 파일로 교체(동일 파일명).
- `index.html`: 섹션별 내용 수정.
- `styles.css`: 색상/폰트/간격 등 스타일 수정.

## 섹션 안내
- 소개, 하이라이트, 프로젝트, 연구/세미나, 수상 & 활동, 기술/과목, 연락처
- 필요하면 블로그/게시글 섹션을 추가하여 포스트 링크를 걸어도 됩니다.

## 로컬 프리뷰(선택)
더블클릭으로 `index.html`을 열어도 되며, 간단한 HTTP 서버를 쓰려면:

```bash
python -m http.server 8000
# http://localhost:8000
```
